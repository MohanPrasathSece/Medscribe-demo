# MedScribe - AI-Powered Medical Transcription

MedScribe is an AI-powered tool designed to transform voice into structured medical documentation. It provides fast, secure, and accurate transcription services for doctors, clinics, and hospitals.

## Features

- **Voice to Text**: Convert spoken medical notes into text.
- **AI Transcription**: Utilizes advanced AI for high accuracy.
- **Secure**: HIPAA compliant documentation.

## Tech Stack

- **Vite**
- **TypeScript**
- **React**
- **shadcn-ui**
- **Tailwind CSS**

## Getting Started

To get started with this project locally, follow these steps:

### Prerequisites

- Node.js & npm installed

### Installation

1. Clone the repository:
   ```sh
   git clone <YOUR_GIT_URL>
   ```

2. Navigate to the project directory:
   ```sh
   cd voice-docs-ai-main
   ```

3. Install dependencies:
   ```sh
   npm install
   ```

4. Start the development server:
   ```sh
   npm run dev
   ```

## Development

- **Build**: `npm run build`
- **Lint**: `npm run lint`
- **Preview**: `npm run preview`
